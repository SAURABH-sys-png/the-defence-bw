## 🎯 Overview

**Defense Eligibility Calculator & SSB Prep Hub** is a modern web platform designed specifically for Indian Armed Forces aspirants.

The platform combines:

✅ Advanced Defense Entry Eligibility Calculations  
✅ Comprehensive SSB Interview Preparation  
✅ Psychological Test Mastery Resources  
✅ Officer Like Qualities (OLQ) Development Guidance  
✅ Curated Story Libraries for PPDT & TAT Practice  

Whether you're preparing for **NDA**, **CDS**, **AFCAT**, or other defense entries, this platform helps you stay informed, prepared, and mission-ready.

---

## 🧮 Defense Eligibility Calculator

A highly accurate eligibility engine that evaluates candidates across multiple defense entries using dynamic eligibility logic.

### 🔍 Eligibility Parameters

- Exact Date of Birth (DOB) calculations
- Gender-specific criteria validation
- Graduation status verification
- Qualification-based filtering
- Multi-entry eligibility matching
- Age cut-off matrix processing
- Real-time eligibility assessment

### 🎖 Supported Defense Entries

- NDA (National Defence Academy)
- CDS (Combined Defence Services)
- AFCAT (Air Force Common Admission Test)
- Additional future entry schemes

### ⚡ Smart Dynamic Matrix Logic

Unlike basic calculators that only estimate age, this platform:

- Computes exact age against official cut-off dates
- Validates qualification requirements
- Applies entry-specific eligibility rules
- Supports multiple eligibility scenarios simultaneously
- Minimizes chances of incorrect eligibility assumptions

### 📊 Physical Standards Reference

Includes detailed physical standards charts covering:

- Height requirements
- Weight standards
- Medical considerations
- Service-specific guidelines
- Gender-based standards where applicable

# 🚀 Getting Started

## 📋 Prerequisites

Make sure you have installed:

- Node.js
- npm

---

## 🔧 Installation

### Clone the Repository

```bash
git clone https://github.com/SAURABH-sys-png/the-defence-bw.git

### Navigate into the Project

```bash
cd the-defence-bw
```

### Install Dependencies

```bash
npm install
```

---

## ▶️ Run Development Server

```bash
npm start
```

Application will be available at:

```text
http://localhost:5173
```

---

## 📦 Production Build

```bash
npm run build
```

Build artifacts will be generated inside the production build directory.
