**The Golden Rule**: 1 minute per question maximum
- Don't get stuck on one question

## Topic-Wise Shortcut Techniques

### 1. Trigonometry (Time Saver)

**Quick Trick for Angle Problems**:
```
If angle questions seem complex, try to convert to 
standard angles (30°, 45°, 60°)
sin²θ + cos²θ = 1 always
```

**Shortcut**: For compound angle questions, use visual representations instead of formulas when possible.

### 2. Algebra Quick Methods

**Solving Quadratic Equations Faster**:
- Use Vieta's formula instead of quadratic formula when sum/product is needed
- For (x-a)(x-b) = 0, roots are directly a and b

**Arithmetic & Geometric Progression**:
- Don't memorize all formulas
- Derive nth term quickly from the pattern
- For GP with common ratio r: Sum = a(1-r^n)/(1-r)

### 3. Permutation & Combination Power

**Key Insight**: 
- nCr = nC(n-r)
- If n is large, use this property to reduce calculations

**Permutation Trick**:
- For circular arrangements: (n-1)!
- For restricted permutations: Use subtraction method

### 4. Geometry Game Changers

**Coordinate Geometry**:
- Always place shapes at origin or convenient coordinates
- Distance formula: √[(x₂-x₁)² + (y₂-y₁)²]
- Save time by choosing coordinates wisely

**Triangle Properties You Must Know**:
- Area = (1/2) × base × height
- For triangle with vertices: Use determinant method
- Centroid divides median in 2:1 ratio

### 5. Logarithm Shortcuts

**Basic Conversions**:
- log(ab) = log(a) + log(b)
- log(a/b) = log(a) - log(b)
- log(a^n) = n·log(a)

**Speed Tip**: Recognize common logarithm patterns immediately

## Problem-Solving Strategies

### Strategy 1: Estimation Before Calculation
- Look at answer options first
- Estimate the answer range
- This helps in quick elimination

### Strategy 2: Graphical Approach
- For functions, quickly sketch the graph
- Identify behavior without detailed calculation
- Faster for limit and continuity questions

### Strategy 3: Backwards Substitution
- When you have multiple-choice answers
- Plug answers back into the equation
- Check which one satisfies the condition

### Strategy 4: Pattern Recognition
- In sequences, identify the pattern first
- Save time by recognizing types (AP, GP, etc.)
- General formulas become obvious from patterns

## Negative Marking Strategy

### The 0.25 Deduction Rule
- Only attempt if you're 80% confident
- Leave questionable problems
- Remember: Skipping is better than wrong answer

### Safe Zone Approach
- Attempt questions from your strong topics first
- Build confidence with easy wins
- Use remaining time for tricky questions

## Practice Problems

### Sample Problem 1: Quick Algebra
If 2x + 3y = 12 and 3x - 2y = 5, find x + y

**Quick Solution**: Add equations → 5x + y = 17
From first equation: y = (12 - 2x)/3
Substitute and solve... (30 seconds max)

### Sample Problem 2: Trigonometry Speed
Find sin(75°)

**Quick Method**: sin(75°) = sin(45° + 30°)
= sin45°cos30° + cos45°sin30°
= (√2/2)(√3/2) + (√2/2)(1/2)
= (√6 + √2)/4
